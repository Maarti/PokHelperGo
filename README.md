# PokHelperGo
Helper for Pokemon GO

## Installation

``
gcc `pkg-config gtk+-3.0 --cflags` PokHelperGo.c -o PokHelperGo `pkg-config gtk+-3.0 --libs`
``

## Usage

### Pidgey Grinding Part

Enter your number of Pidgey then your number of candies then submit.
